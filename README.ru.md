# Kebab Chefs! - Menu Limit Remover

Простой мод для [MelonLoader](https://melonwiki.xyz/) для игры **Kebab Chefs! - Restaurant Simulator**, убирающий стандартное ограничение в 10 блюд в меню ресторана — теперь вы (и ваши друзья, если играете по сети) можете добавить в меню сколько угодно блюд.

> ⚠️ перед установкой любого мода рекомендуется делать бэкап сохранений.

## Возможности

- Увеличивает максимальное количество блюд, которое можно добавить в меню ресторана (в оригинале лимит — 10).
- Работает как в одиночной игре, так и в мультиплеере (синхронизировано через Netcode).

## Требования

- **[MelonLoader](https://melonloader.com/)** — мод **не** работает сам по себе, сначала обязательно нужно установить MelonLoader.

## Установка

1. **Установите MelonLoader**, если ещё не сделали этого:
   - Скачайте установщик MelonLoader с [melonloader.com](https://melonloader.com/).
   - Укажите путь к `Kebab Chefs! - Restaurant Simulator.exe`.
   - Запустите игру один раз, чтобы MelonLoader завершил первоначальную настройку (создадутся папки `Mods`, `Plugins` и `UserData` в папке игры).
2. **Скачайте мод**: возьмите последнюю версию `.dll`-файла со страницы [Releases](https://github.com/PhantomUnk/KebabChefs-UnlimitedMenu/releases) этого репозитория.
3. **Перенесите `.dll`-файл** в папку `Mods` внутри папки установки игры (та же папка, где лежит `.exe` игры).
4. Запустите игру как обычно. В консоли MelonLoader должна появиться строка о том, что мод загрузился.

## Важно про мультиплеер

Мод может быть установлен **ТОЛЬКО у хоста** — тогда всё будет работать. Если мод будет и у хоста, и у клиента, работать **ТОЖЕ** будет.

## Решение проблем

- **Если у вас пиратская версия игры** — закиньте `winmm.dll` из вашего онлайн-репака.
- **Мод вообще не появляется в логе** — убедитесь, что `.dll`-файл лежит прямо в папке `Mods` (не во вложенной папке), и что MelonLoader был установлен корректно.
- **Конфликт с другими инструментами, которые тоже подключаются через proxy-DLL** (некоторые сторонние патчи запуска или инструменты обхода защиты тоже используют библиотеку с типовым именем для подключения к игре, например `winmm.dll`, `version.dll`, `dinput8.dll`). Сам MelonLoader использует `version.dll`. Если вы используете другой инструмент с тем же именем файла — реально загрузится только один из них; переименуйте или восстановите нужный файл так, чтобы оба инструмента использовали разные имена файлов (если это поддерживается тем инструментом).
- Логи находятся в `MelonLoader/Logs/Latest.log` в папке игры — если что-то не работает, сначала загляните туда.

## Сборка из исходников

Это патч через [Harmony](https://harmony.pardeike.net/), собранный на основе Il2Cpp interop-сборок игры (которые MelonLoader генерирует локально при первом запуске). Чтобы собрать самостоятельно:

1. Установите [MelonLoader.VSWizard](https://github.com/TrevTV/MelonLoader.VSWizard) для Visual Studio.
2. Создайте новый проект "MelonLoader Mod" и укажите `.exe` вашей игры — нужные ссылки на библиотеки подключатся автоматически.
3. Добавьте в проект `Core.cs` из этого репозитория.
4. Соберите проект, затем скопируйте получившийся `.dll` в папку `Mods`.

## Дисклеймер

Мод только патчит значения, уже существующие в коде игры, во время выполнения (через Harmony) — он не распространяет, не декомпилирует для republishing и не изменяет оригинальные файлы игры на диске. Для работы требуется рабочая копия игры.
